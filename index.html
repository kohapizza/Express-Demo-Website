Get started

Requirements

  • GHC 9.x
  • Stack latest
  • Juman++ Japanese tokenizer
  • KNP Japanese parser

Build & run

# Clone
git clone https://github.com/DaisukeBekki/lightblue
cd lightblue

# Build (first run takes a while)
stack build

# Launch Express
stack exec express

# Open http://localhost:3000

Full setup instructions (Juman++, KNP, dictionary paths): see README on GitHub.
